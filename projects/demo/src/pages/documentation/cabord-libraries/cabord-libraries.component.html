Cabord is a collection of libraries. Explore them and learn how to install and use them.

  • @cabord/core

    It is the main framework-agnostic package. It can be used alone in vanilla JavaScript project.

    Learn more about this library in "Core Concepts" section.

  • @cabord/angular

    The Angular package. It contains ready-to-use angular apapter.

    Learn more about this library in "Angular" section.

  • @cabord/react

    The React-specific library. It provides a convenient way to use cabord in react.

    Learn more about this library in "React" section.

  • @cabord/realtime-plugin

    The library to provide realtime collaboration.

    Learn more about this library in "Realtime Plugin" section.
