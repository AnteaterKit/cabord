Cabord is a collection of libraries, built with TypeScript. It helps you to create an whiteboard application with simple javasricpt/typescript and with populal front-end frameworks: angular, react.

The libraries provide you the flexible API to create infinite whiteboard with a set of primitives: pen, rect, cycle, etc.

Motivation to use Cabord

  • Cabord compatible with populal front-end frameworks: angular, react.
  • Cabord has a set with most popular primitives: pen, rect, cycle, etc.
  • Cabord is Infinite whiteboard.
  • Cabord has a realtime providers based on popular websocket tools.
  • Cabord is robust. The whole project is developed with strict TypeScript mode. Our code is covered by hundreds of Cypress tests.
  • Cabord is framework-agnostic package.
